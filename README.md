# Cadastro de Eventos

## Pre requisito

- Java 8
- JSF
- CDI
- Primefaces
- Maven
- Tomcat 9.0

## Clone o projeto

```sh
git https://github.com/Gian-f/ejp.git
```

## Iniciar o projeto

```sh
Adicione o projeto ao servidor configurado e start a aplicação
```

## Rodar no Browser

```sh
http://localhost:8080/ejp/CadastrarCliente.xhtml
```
